New Here?

Hey! Welcome to our lil' system. Here's some things you should really, really do if you haven't already -- it'll make things run far more smoothly, so we can all concentrate on Scavhunt, not on who's doing what:

  1. Fill out our survey so we know when you're available and what cool stuff you can do.
  2. Add yourself to the database. Then we can use our software to track who's doing what and how we're doing. Maybe that way no one will call you at 4 am wanting to know if the giant catapault is finished.
  3. Consider skimming the database manual to get an idea of what you can do.
  4. Go play around in the database. It's got last year's data in it (or will until Wednesday night).

Logging In

First, log into the database, using your personal database username and password, here.

If you don't have a username yet, you can create one for youself here. Please read the directions below first, though.

The database user list was accidentally reset partway through ScavHunt last year, so if you haven't set your password since then, your password is blank.

Updating Your User Info

Please update your user info. Your User Page is the first page you are sent to after you log into the database. If you are already inside the database, you can get to your User Page by going to the Search Page and performing a User Search for yourself.

Once you are at your User Page, you can edit your user info. You need to enter at least a username ("login handle"), one piece of contact information, and a personal password (which needs to be entered twice for verification purposes) in order for the update to go through. If you already have some of these data entered, they will be filled into the form automatically for you. If there is no field for a "login handle" on your User Page, you already have a username, which cannot be changed.

If your attempt to update your user info does not go through (probably because you have not entered all of the necessary information, as listed above), you will be thrown to the Add User page. Do not fill out this page unless you want to create a new database user account; simply press the Back button in your browser to try updating your user info again.

Once you have correctly updated your user info, the new version of your User Page will be displayed.

Using the Database

After you log into the database, you will be presented with your User Page, which has a link to the Database Search Page at the top. Alternately, you can go there directly from the link provided here.
